\section{Метод решения}
Данная программа реализует многопроцессную обработку текстовых данных с помощью
shared memory:
Ключевые компоненты:\\
• Shared memory − Linux\\
Системные вызовы:\\
• Windows: CreatePipe, CreateProcess, ReadFile, WriteFile\\ 
• Linux: pipe, fork, excel, read, write\\
Программа обеспечивает кроссплатформенность и четкое разделение ответственности
между модулями.
